MyFlix

The objective of this project is to build the server-side component of a "movies" web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

End Points

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all movies /movies GET None A JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title /movies/[title] GET None A JSON object holding data about the particular movie requested
Return data about a genre (description) by name/title /movies/genre/[genreName] GET None A JSON object holding data about the genre (description)
Return data about a director (bio, birth year, death year) by name /movies/director/[directorName] GET None A JSON object holding data about the director (bio, birth year, death year)
Allow new users to register /users/[id] PUT A JSON object holding data about the user (name, favorite movies) A JSON object holding data about the user who was added (id, name, favorite movies)
Allow users to update their user info (username) /users PUT A JSON object holding the user ID and new username A JSON object holding data about the updated user (id, name, favorite movies)
Allow users to add a movie to their list of favorites /users/[id]/[movieTitle] POST None Returns a message saying "[movieTitle] has been added to user [id]'s array"
Allow users to remove a movie from their list of favorites /users/[id]/[movieTitle] DELETE None Returns a message saying "[movieTitle] has been removed from user [id]'s array"
Allow existing users to deregister /users/[id] DELETE None Returns a message saying "[movieTitle] has been removed from user [id]'s array"
